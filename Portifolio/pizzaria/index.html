Projeto Pizzaria

Projeto Pizzaria é meu primeiro projeto responsivo com mais de uma página, porém estático. O mesmo possui tela de início, login, criar conta, visualizar perfil, tela home, e ver produto. Esse projeto foi vital para eu desenvolver páginas funcionais, eficientes e com uma boa estética.

Segue abaixo algumas das páginas:

Tela Home

A tela home mostra as opções de pizza, o preço, e também um campo para pesquisa.

Tela do Produto

A tela do produto mostra a descrição, o preço e as avaliações de alguma pizza em específico.

Tela Criar Conta

A tela de criar conta possui campos de preenchimento para que o cliente possa entrar no site com seus dados.
